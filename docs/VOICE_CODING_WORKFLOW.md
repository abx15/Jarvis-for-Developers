# Voice Coding System - Workflow Documentation

## Overview

The Voice Coding System allows developers to control the AI Developer Operating System using natural voice commands. This document explains the complete workflow and provides examples.

## Architecture

```
Microphone Input → Speech to Text → Voice Command Parser → AI Agents → Code Generation
```

## Voice Pipeline

### 1. Speech Recognition
- **Frontend**: Web Speech API via `useVoice` hook
- **Backend**: Voice Engine Service (port 8002)
- **Real-time**: Live transcription with interim results

### 2. Command Processing
- **Service**: `VoiceService` in `services/voice_service.py`
- **Intent Detection**: Pattern matching for command types
- **Entity Extraction**: Files, languages, APIs, bugs

### 3. Agent Execution
- **Router**: Maps intents to appropriate AI agents
- **Agents**: Code, Debug, Refactor, Test, Search
- **Context**: Repo ID, file context, parameters

## Voice Command Types

### 1. Generate Code
**Keywords**: create, generate, make, write, build, implement, develop

**Examples**:
- "Create a REST API for products"
- "Generate a Python function to calculate factorial"
- "Make a React component for user profile"
- "Build a login page with authentication"

**Entities Extracted**:
- `file_name`: Target file name
- `language`: Programming language
- `api`: True for API requests

## Example Workflows

### Example 1: Creating a REST API

**Voice Command**: "Create a REST API for products"

**Processing Steps**:
1. **Speech to Text**: "create a rest api for products"
2. **Intent Detection**: `generate_code`
3. **Entity Extraction**:
   - `api`: True
   - `resource`: "products"
   - `type`: "api"
4. **Agent Selection**: Code Agent
5. **Task Preparation**: "Create a REST API for products\n\nContext: Type: API, Resource: products"
6. **Execution**: Code agent generates FastAPI endpoints

**Expected Output**:
```python
# Generated by AI Agent
from fastapi import FastAPI, HTTPException
from pydantic import BaseModel

app = FastAPI()

class Product(BaseModel):
    name: str
    price: float
    description: str

products = []

@app.get("/products")
async def get_products():
    return products

@app.post("/products")
async def create_product(product: Product):
    products.append(product)
    return product

@app.get("/products/{product_id}")
async def get_product(product_id: int):
    if product_id >= len(products):
        raise HTTPException(status_code=404, detail="Product not found")
    return products[product_id]
```
